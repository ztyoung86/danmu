# danmu

本项目 fork 自[littlecodersh/danmu](https://github.com/littlecodersh/danmu),目前仅做了少量修改以做测试。

仅支持斗鱼TV和熊猫直播。

by Albert Yang

## *nix 系统运行方法

```bash
source venv/bin/activate
python run.py <douyu | panda> <room_id>
```

## Windows 系统运行方法

1. 安装 python 运行环境

2. 安装 requests 包
   ```bash
   pip install requests
   ```

3. 运行
   ```bash
   python run.py <douyu | panda> <room_id>
   ```
